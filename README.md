Исследование рынка недвижимости Астаны на основе данных с Krisha.kz.

##  Ключевые находки

- Самый дорогой район: **Есильский — 725 000 ₸/м²**, самый доступный: **Байконурский — 476 839 ₸/м²** — разница **+52%**
- Новостройки (≥2015) дороже вторичного жилья на **+42.8%**
- Первый этаж: дисконт **−21.5%**, последний этаж: **−15.1%** от медианы
- Бюджет на 2-комн. (медиана по городу): **~35 млн ₸**
- Лучшая модель **CatBoost**: R² = **0.864**, MAE = **7.41 млн ₸**, MAPE = **16.3%**

##  Данные

- **Источник:** [Kaggle — Astana Real Estate Dataset](https://www.kaggle.com/datasets/turarr/astana-real-estate-dataset)
- **Объём после чистки:** ~4 800 объявлений
